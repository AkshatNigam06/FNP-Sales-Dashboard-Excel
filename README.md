# 📊 FNP Sales Dashboard (Excel)

## 📌 Project Overview
This project is an interactive **FNP Sales Dashboard** built in Microsoft Excel. The dataset was cleaned and transformed using **Power Query Editor**, and the dashboard was created using Pivot Tables, Pivot Charts, Slicers, and KPI cards to provide meaningful business insights.

---

## 🎯 Objective
The goal of this project is to analyze FNP sales data and identify:
- Revenue trends
- Top-performing product categories
- Occasion-wise sales
- City-wise performance
- Monthly sales trends
- Customer spending patterns
- Delivery performance

---

## 🛠️ Tools & Technologies
- Microsoft Excel
- **Power Query Editor**
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting

---

## 🔄 Data Preparation (Power Query Editor)
The raw dataset was cleaned and transformed using **Power Query Editor** by:
- Removing duplicate records
- Handling missing values
- Correcting data types
- Renaming columns
- Creating calculated columns
- Preparing the dataset for dashboard reporting

---

## 📈 Dashboard KPIs
- 💰 Total Revenue
- 📦 Total Orders
- 👤 Average Customer Spend
- 🚚 Average Delivery Time
- 🎁 Revenue by Category
- 🏙️ Revenue by City
- 📅 Revenue by Month
- 🎉 Revenue by Occasion
